# Senior PM Exam Study (Senior-PM-Exam-Study)

系统规划与管理师（项目经理高级）软考备考专业学习系统

## 📚 项目介绍

这是一个**结构化、可持续、与实战结合**的项目经理高级证书学习系统，目标是帮助考生在2026年5月通过软考项目经理高级（系统规划与管理师）。

### 🎯 核心目标

- **短期**：2025年5月高质量通过软考高级（≥75分）
- **中期**：建立完整的项目管理知识体系，服务工作实战
- **长期**：成为懂技术、懂管理、懂业务的复合型AI产品经理

### 📊 考试信息

| 维度 | 说明 |
|------|------|
| **考试级别** | 软考高级（最高等级，难度★★★★★） |
| **考试形式** | 笔试（75分）+ 论文（25分）= 100分总分 |
| **及格线** | 60分 |
| **考试周期** | 每年5月、11月 |
| **考试时长** | 上午2.5小时笔试 + 下午2.5小时论文 |

## 🎯 知识体系（5大模块）

### A. 基础认知与框架 (12%)

项目定义、PMBOK框架、ITSS体系、战略对齐

### B. 项目规划与范围管理 (18%)

需求收集、范围定义、WBS、变更管理

**🔥 高频出题点**：Scope Creep预防、范围基准建立

### C. 进度与资源管理 (20%) ⭐⭐ 最高权重

活动定义、进度估算、关键路径法(CPM)、资源平衡

**🔥 高频出题点**：在约束条件下制定可行计划、浮动时间计算

### D. 质量、成本与采购 (20%)

质量规划、成本预估、挣值管理、采购战略

**🔥 高频出题点**：成本/质量/进度三角形平衡

### E. 风险与沟通管理 (18%)

风险识别、风险分析、风险响应、沟通管理

**🔥 高频出题点**：预见项目风险、干系人冲突处理

### F. 论文答题技巧与实战 (12%)

论文出题类型分析、答题框架、案例素材积累

**🔥 关键**：论文得分25分，直接影响最终成绩

## 📅 学习规划

### 推荐学习顺序

1. **第一阶段（60%时间）** - 核心编排模式
   - 重点：C（进度与资源）+ D（质量、成本、采购）+ E（风险与沟通）
   - 为什么：这3个模块权重最高（58%），最容易出论文题

2. **第二阶段（20%时间）** - 范围与论文技巧
   - 重点：B（范围管理）+ F（论文技巧）
   - 为什么：范围决定后续，论文直接影响分数

3. **第三阶段（15%时间）** - 基础巩固与真题
   - 重点：A（基础框架）+ 历年真题模拟
   - 为什么：查漏补缺 + 适应考试节奏

4. **冲刺阶段（5%时间）** - 高频错题与心理建设
   - 高频错题复习 + 论文框架磨练

## 💬 学习方式

### 核心学习循环（每个知识点）

```
诊断 → 解释+类比 → 实践+题目 → 反思+错题分析
```

1. **诊断** - 我先问你对这个考点的直觉理解和实战经历
2. **解释** - 我用类比和你的项目例子讲解核心逻辑
3. **实践** - 做题、写论文、验证理解
4. **反思** - 分析错题、梳理漏洞

### 我的教学特点

✅ **苏格拉底式引导** - 不直接给答案，而是通过提问引导你发现

✅ **实战导向** - 既讲"考试怎么考"，也讲"项目里怎么用"

✅ **零容忍幻觉** - 必须查证、禁止编造、宁缺毋滥

✅ **活跃反驳** - 欢迎你的批判性思维和不同意见

✅ **进度透明** - 每次会话都更新学习记录，进度可追踪
